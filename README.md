## 算法步骤

```
1. 计算整张图片的均值
2. 计算池化块的均值
3. 计算池化块的最大值
4. if 若池化块的均值 > 图片的整体均值
      该池化点为池化块的均值
   else:
      该池化点为池化块的最大值
```

## 算法思想 
  对于图片中平坦地区进行模糊化，对图片中的边缘地区进行尖锐化。
